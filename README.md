# Traffic Monitor (GlassWire Style)

GlassWireにインスパイアされた、リアルタイムでネットワークトラフィックを監視し、アプリケーションごとの通信状況を可視化するデスクトップツールです。

## 🚀 主な機能

- **リアルタイムトラフィック表示**: ダウンロードとアップロードの速度をリアルタイムでグラフ表示（Chart.js）。
- **アプリケーション監視**: 通信を行っているプロセスの名前、PID、リモートホストのIPアドレス、ポート、接続状態を一覧表示。
- **GeoIP（位置情報）取得**: 接続先のIPアドレスから国名を自動的に割り出し、表示します。
- **スマートアラート機能**:
  - **新規アプリケーション検知**: 起動後に初めてネットワーク通信を行ったアプリを即座に通知。
  - **DNS変更検知**: システムのDNSサーバー設定が変更された場合にアラートを表示。
- **デスクトップアプリ形式**: Electronを使用し、ブラウザを開かずに独立したアプリとして動作。

## 🛠 テクノロジースタック

- **Frontend**: HTML5, CSS3, JavaScript, Chart.js, Socket.io-client
- **Backend**: Node.js, Express, Socket.io
- **Infrastructure**: Electron
- **Data Source**: systeminformation (OS統計取得), ip-api.com (GeoIP)

## 📦 セットアップと実行方法

### 前提条件
- [Node.js](https://nodejs.org/) (v16以上推奨) がインストールされていること。

### インストール
1. プロジェクトディレクトリに移動します。
   ```powershell
   cd traffic_monitor
   ```
2. 依存パッケージをインストールします。
   ```powershell
   npm install
   ```

### アプリの起動
以下のコマンドを実行すると、デスクトップウィンドウが立ち上がります。

```powershell
npm start
```

> **重要: 管理者権限での実行を推奨**  
> 全てのプロセスの名前や詳細情報を取得するためには、ターミナル（PowerShell等）を「管理者として実行」した状態で `npm start` を行ってください。

## 📝 注意事項

- **GeoIP API**: 外部の無料API (`ip-api.com`) を使用しています。短時間に大量のリクエストを送ると制限がかかる場合がありますが、本アプリではキャッシュ機能を搭載して負荷を軽減しています。
- **セキュリティ**: 本アプリはローカルネットワークの監視のみを目的としています。収集したデータが外部に送信されることはありません（GeoIP取得のためのIPアドレス送信を除く）。
